Hangman

Understand the Problem

  • Player one selects a word
  • Player one draws the board
  • Player two guesses one letter at a time
  • For each correct guess all of those letters are added into the word
  • For each incorrect guess one piece is added to the hangman
  • Player two continues guesses until word is filled out or full body is drawn

    • Pseudocode the High Level Approach

    
Game setup
Select a word
Create the board
Start the game loop
Opposing player guesses letters
Write correct letter in given space
Letter incorrect
Draw one part of figure
If the game should end
Display the proper victory
Else
Keep looping with new letters until phrase or
figure completed
